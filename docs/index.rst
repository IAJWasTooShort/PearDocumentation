Documentation
==================

This is the new Official PearOS documentation. It has all the information you need about PearOS: A Linux distribution inspired by macOS. This documentation is currently a work in progress, and if there is any info that is not on here that you think should be added please ask IAJ on discord. 

PearOS
==================

PearOS is a Linux Distibution inspired by Apple's macOS. It features a clean UI, and apps that can be personalised to your liking.
